## prototype

```jsx
const user = {
	name : 'Mike;
}
console.log(user.name) // "Mike"
user.hasOwnProperty('name') // true
user.hasOwnProperty('age') // false

-> hasOwnProperty() 함수는 만들지 않았는데 어떻게 사용하는거지?
```

객체에서 프로퍼티를 읽으려고 할 때 없으면 프로토타입에서 찾는다.

```jsx
const user = {
	name : 'Mike;
	hasOwnProperty : function(){
			console.log('haha');
	}
}
user.hasOwnProperty() // haha

-> 만약 객체 프로퍼티에서 찾으면 탐색을 멈춘다.
```

프로퍼티를 좀 더 이해하기 위해 상속개념을 배워보자.

중복되는 코드가 많은 경우

```jsx
const bmw = {
	color: "red",
	wheel: 4,
	navigation : 1,
	drive(){
		console.log("drive...");
	},
}

const benz = {
	color: "black",
	wheel: 4,
	navigation : 1,
	drive(){
		console.log("drive...");
	},
}

const audi = {
	color: "blue",
	wheel: 4,
	navigation : 1,
	drive(){
		console.log("drive...");
	},
}

```

프로토타입 사용 

```jsx

const car = {
	wheels : 4,
	drive(){
		console.log("drive");
	},
}

const bmw = {
	color: "red",
	wheel: 4,
	navigation : 1,
}

bmw.__proto__ = car;

const x5 = {
	color: "white",
	name: "x5"
};

x5.__proto__ = bmw;

다른 사용법
```
